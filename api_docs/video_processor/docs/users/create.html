Request new user token

ID: users/create
Register new user and obtain api_token. When a new user starts using a mobile app, the app sends a request to the server for a unique key which is used to subscribe user requests to identify user.

Request

URL
/api/v1/users.json
Method
POST
Format
json
Headers
Name Value Description Required
Url params
Data params

Responses

Success
HTTP Status
201 CREATED
Format
json
Data params
{
api_token: string // Auth token; example value: ikbn]zayolrS^Mcd
}
Sample Call
curl -v -H "Accept: application/json" -H "Content-type: application/json" -X POST http://video_processor.dev/api/v1/users.json
Sample Response
{
   "api_token":"YwXdQ64vI5oam8ukfUx4SAtt"
}
