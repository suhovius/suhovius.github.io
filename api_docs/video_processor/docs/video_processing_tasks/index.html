Get video processing tasks list

ID: video_processing_tasks/index
Returns video processing tasks list array ordered by created at in descending order. User can see the list of the requests with their statuses (done, failed, scheduled, processing).

Request

URL
/api/v1/video_processing_tasks.json
Method
GET
Format
json
Headers
Name Value Description Required
Authorization: Token token=:token_value :token_value - is an authorization token value (api_token) Yes
Url params
{
page: integer, // Pagination page; default value: 1
per_page: integer // Pagination place per page; default value: 25
}
Data params

Responses

Success
HTTP Status
200 OK
Format
json
Data params
[
  // document
{
id: string, // Video processing task database ID; example value: a8fd48e26441f8d54c88a47d
trim_start: integer, // Video start trimming parameter in seconds; example value: 2
trim_end: integer, // Video end trimming parameter in seconds; example value: 21
source_video:  // document, Source video file details
{
url: string, // source file url; example value: http://video_processor.dev/system/video_processing_tasks/source_videos/57d4/6ce8/07ae/1d3b/2b95/ad50/original/test_video.mov?1473539304
duration: integer // source file duration; example value: 85
}
,
result_video:  // document, Result video file details
{
url: string, // result file url; example value: http://video_processor.dev/system/video_processing_tasks/result_videos/36d4/6ce8/07ae/1d3b/2b75/ad50/original/test_video.mov?1773499304
duration: integer // result file duration; example value: 2
}
,
started_at: timestamp, // Task started at unix timestamp
completed_at: timestamp, // Task completed at unix timestamp
failed_at: timestamp, // Task failed at unix timestamp
created_at: timestamp, // Task created at unix timestamp
updated_at: timestamp, // Task updated at unix timestamp
state: string, // Task state. Could be: scheduled, processing, done, failed; example value: failed
last_error: string // Task last error message that happened during video processing; example value: FFMPEG: Failed encoding
}
]
User is not authorized
HTTP Status

Format
json

error: string // Api Token is not valid error message; example value: Bad Credentials
}
